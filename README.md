## 📌 Overview

This project implements a complete **end-to-end NLP pipeline** for analyzing multilingual social media data.
It covers data generation, preprocessing, feature extraction, visualization, and deep learning-based sentiment analysis, along with an interactive dashboard.

---

## 🚀 Features

* 📊 Exploratory Data Analysis (EDA)
* 🧹 Text Preprocessing (cleaning, normalization, tokenization)
* 🌐 Multilingual Data Handling (English, Hindi, Telugu, Tamil, etc.)
* 🔤 Feature Extraction:

  * TF-IDF
  * Word2Vec (Gensim)
* 📉 Dimensionality Reduction:

  * PCA
  * t-SNE visualization
* 🤖 Deep Learning Model:

  * Sentiment Analysis
  * mBERT-based classification
* 📊 Interactive Dashboard for visualization

---

## 🛠️ Tech Stack

* **Language:** Python
* **Libraries:**

  * Pandas, NumPy
  * Scikit-learn
  * Gensim
  * Matplotlib, Seaborn
  * TensorFlow / PyTorch
  * Transformers (mBERT)

  * Telugu

  * Marathi
  * Punjabi
  * Malayalam
  * Tulu

## 📈 Model Details

* **Embeddings:**

  * TF-IDF for baseline
  * Word2Vec for semantic representation
* **Dimensionality Reduction:**

  * PCA for linear projection
  * t-SNE for visualization
* **Deep Learning:**

  * mBERT for multilingual classification
  * Sentiment analysis model
